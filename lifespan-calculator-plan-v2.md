# Lifespan Calculator (남은 수명 계산기) — 서비스 기획서

---

## 1. 컨셉 한 줄 요약

**"당신에게 남은 시간을 알려드립니다"**

생년월일, 생활습관, 건강 정보를 입력하면 통계 기반 예상 수명과 남은 시간을 실시간 카운트다운으로 보여주는 바이럴 웹 서비스.

---

## 2. 핵심 심리 메커니즘

수명 계산기가 바이럴 되는 이유는 **죽음에 대한 보편적 호기심 + 결과의 자극성** 때문.

- "나 87세래" → 안도 → 공유
- "나 68세래" → 충격 → "이거 실화냐" 공유
- 어떤 결과든 공유 동기가 발생하는 구조

추가로 **실시간 카운트다운**이 시각적 임팩트를 만듦. "남은 시간: 47년 3개월 12일 7시간 23분 41초... 40초... 39초..." 초가 줄어드는 걸 보는 순간 스크린샷 or 화면 녹화로 SNS 공유.

---

## 3. 입력 항목 설계

인생 영수증과 일부 겹치므로 통합 사이트 시 데이터 재사용 가능. **필수 4개 + 건강 습관 8개 + 정신건강 3개**, 총 30초 내 완료 목표.

### 필수 입력

- **생년월일**: 날짜 선택. 기준 연령 산출.
- **국가**: 드롭다운 (IP 자동 선택). 국가별 기대수명 기준값.
- **성별**: 2버튼 선택. 남녀 기대수명 차이 (전 세계 평균 약 5년).
- **키/체중**: 숫자 입력 (cm/kg or ft/lb). BMI 계산 후 수명 보정.

### 생활습관 입력 (슬라이더 / 버튼 선택)

- **흡연**: 4버튼 (안함 / 과거 / 가끔 / 매일). 흡연자 평균 -10년 (CDC).
- **음주**: 4버튼 (안함 / 가끔 / 주2-3 / 매일). 과음 -5년, 적정음주 +/-0.
- **운동 빈도**: 슬라이더 (주 0-7회). 규칙적 운동 +3-5년 (WHO).
- **수면 시간**: 슬라이더 (4-12시간). 6시간 미만 or 9시간 초과 시 -2-3년.
- **식습관**: 3버튼 (건강식 / 보통 / 패스트푸드). 지중해식 +4-7년 (NEJM).
- **운전 습관**: 3버튼 (안함 / 보통 / 장거리통근). 교통사고 리스크 반영.
- **좌식 시간**: 슬라이더 (하루 2-14시간). 8시간+ 좌식 -2년.
- **자외선 노출**: 3버튼 (실내직 / 보통 / 야외직). 피부암 리스크 등.

### 정신건강/사회적 요인

- **스트레스 수준**: 슬라이더 (1-10). 만성 스트레스 -2-3년.
- **사회적 관계**: 3버튼 (활발 / 보통 / 고립). 사회적 고립 -5년 (Holt-Lunstad 메타분석).
- **결혼/파트너 유무**: 2버튼. 기혼 남성 +3년, 여성 +1.5년 (통계).

### 가족력 (선택, 접힌 상태)

- **부모 생존 여부/사망 나이**: 숫자 입력. 유전적 수명 보정 +/-3-5년.
- **가족 내 주요 질환**: 멀티 선택 (심장병 / 암 / 당뇨 / 없음). 리스크 가중치 적용.

가족력은 "더 정확한 결과 받기" 아코디언으로 숨김. 입력 안 하면 국가 평균으로 계산.

## 4. 수명 계산 로직

### 4-1. 기본 구조

    예상 수명 = 국가별 기대수명(성별) + SUM(각 요인별 보정값)

### 4-2. 국가별 기대수명 기준 (WHO 2023 데이터)

- 한국: 남 80.5 / 여 86.5
- 일본: 남 81.5 / 여 87.6
- 미국: 남 74.8 / 여 80.2
- 영국: 남 79.0 / 여 82.9
- 브라질: 남 72.0 / 여 79.4
- 인도: 남 69.5 / 여 72.2
- 전 세계 평균: 남 70.8 / 여 75.9

200개국+ 데이터를 JSON으로 내장. WHO Life Tables 기반.

### 4-3. 보정 요인별 가중치 (년 단위)

**수명을 줄이는 요인 (최악 케이스)**

- 흡연 (매일): -10.0년 (CDC, BMJ)
- 음주 (매일 과음): -5.0년 (Lancet 2018)
- BMI 비만: -5.0년 (NEJM)
- 사회적 고립: -5.0년 (Holt-Lunstad)
- 운동 안 함: -3.0년 (WHO, JAMA)
- BMI 저체중: -3.0년 (NEJM)
- 수면 부족/과다: -3.0년 (Sleep Foundation)
- 식습관 불량: -3.0년 (NEJM)
- 스트레스 극심: -3.0년 (APA)
- 좌식 8시간+: -2.0년 (Lancet)
- BMI 과체중: -2.0년 (NEJM)
- 가족력 단명: -3.0년 (Twin studies)

**수명을 늘리는 요인 (최선 케이스)**

- 규칙적 운동: +4.5년
- 건강한 식습관: +4.0년
- 결혼/파트너: +1.5~3.0년
- 가족력 장수: +3.0년
- 활발한 사회관계: +2.0년
- 적정 수면: +1.0년
- 낮은 스트레스: +1.0년
- 적은 좌식: +1.0년

### 4-4. 보정값 상한/하한

- 총 보정값 상한: +15년 (비현실적 결과 방지)
- 총 보정값 하한: -20년
- **최소 예상 수명: 현재 나이 + 5년** (윤리적 하한선)
- **최대 예상 수명: 국가별 기대수명 + 20년**

하한선이 핵심. "당신은 내년에 죽습니다" 같은 결과는 절대 안 됨. 하한선을 넉넉히 잡아서 불쾌하지 않으면서도 자극적인 범위 유지.

### 4-5. 결과에 랜덤성 부여

- 동일 입력이어도 +/-0.5~1.5년 랜덤 변동 추가
- 이유: "다시 해봐도 똑같네" 하면 재방문 안 함. 약간의 변동이 있어야 "어? 아까랑 다르다" → 재시도
- 단, 변동폭이 커서 신뢰도를 해치면 안 됨

## 5. 결과 페이지 설계

### 5-1. 결과 표시 구성

메인 영역 (첫 화면):

    YOUR LIFE CLOCK / 당신의 생명시계

    예상 수명: 84세 (2079년)

    남은 시간
    47년 3개월 12일
    07:23:41  <-- 실시간 감소

    [======||||||||] 56% 사용
    (프로그레스 바: 인생 진행률)

    당신의 남은 계절
    봄 47번 / 여름 47번 / 가을 47번 / 겨울 47번

    당신의 남은 주말: 2,452일
    당신의 남은 크리스마스: 47번
    당신의 남은 생일 케이크: 47개

### 5-2. 핵심 결과 항목

**메인 (즉시 눈에 들어오는 것)**

- **예상 수명**: 84세 (2079년). 핵심 결과.
- **남은 시간 카운트다운**: 47년 3월 12일 07:23:41. 초가 줄어드는 시각적 충격.
- **인생 프로그레스 바**: 56%. 직관적 "이만큼 썼다".

**서브 (스크롤 시 보이는 것 - 감성 자극)**

- **남은 봄/여름/가을/겨울**: 각 47번. "봄이 47번밖에 안 남았어".
- **남은 주말**: 2,452일. "주말이 겨우 이것밖에?".
- **남은 크리스마스**: 47번. 절기/명절의 유한함.
- **남은 생일**: 47번. 가장 개인적인 항목.
- **남은 식사**: 51,465끼. 숫자의 유한함.
- **부모님과 남은 시간**: 약 312일 (연 26일 만남 가정 x 12년). **최강 감성 펀치.**

**"부모님과 남은 시간"이 바이럴 킬러 항목.** 부모 나이를 선택 입력으로 받고, 만남 빈도(매주/매월/명절만)를 곱하면 "부모님과 남은 시간: 약 312일" 같은 결과가 나옴. 이 항목 하나가 감성 공유를 폭발시킴.

### 5-3. 수명 영향 분석 (어떤 습관이 얼마나?)

결과 하단에 바 차트로 시각화:

    당신의 수명에 영향을 준 요인

    [+] 운동 습관     ============  +4.5년
    [+] 식습관        ========      +3.0년
    [+] 사회적 관계   ======        +2.0년
    [+] 수면          ====          +1.0년
    -------------------------------------------
    [-] 흡연          ====================  -10.0년
    [-] 스트레스      ======        -2.0년
    [-] 좌식 생활     ====          -1.5년

이 차트가 "수명 늘리는 법" 콘텐츠로의 자연스러운 브릿지. "운동을 시작하면 +4.5년!" → 건강 콘텐츠 페이지 유도 → 추가 페이지뷰 + 광고.

### 5-4. "수명 늘리기" 시뮬레이션 (인터랙티브)

결과 페이지에서 슬라이더를 조정하면 실시간으로 수명이 변하는 기능:

    [내 수명 늘려보기]

    금연하면?          --*----  수명 +10년!
    운동 주 3회 하면?   ---*---  수명 +3.5년
    7시간 수면하면?     ----*--  수명 +1.5년

    모두 실천하면: 84세 -> 99세! (+15년)

이 시뮬레이션이 재방문 동기를 만듦. "금연하면 10년 늘어난다는데..." → 친구에게 공유 → 재방문. 건강 보험, 헬스케어 앱 제휴 광고의 완벽한 자리.

## 6. UI/UX 설계

### 6-1. 전체 흐름

    [랜딩] -> [입력 폼] -> [로딩] -> [결과: 카운트다운] -> [분석] -> [시뮬레이션] -> [공유]
      1초      20~30초      3초       체류(핵심)          스크롤     인터랙션       바이럴

### 6-2. 랜딩 페이지

- 배경: 검은색, 중앙에 큰 시계 또는 모래시계 아이콘
- 카피: "How much time do you have left?" / "당신에게 남은 시간은?"
- 초가 째깍째깍 줄어드는 카운트다운 애니메이션 (방문자 평균 수명 기준)
- CTA: "내 남은 시간 확인하기" / "Check My Time"
- 하단: "지금까지 OOO명이 확인했습니다" (사회적 증거)

### 6-3. 입력 폼

**스텝 방식 (1/3, 2/3, 3/3)** — 한 화면에 다 보여주면 압도적이므로 단계별로 나눔.

- Step 1: 기본 정보 (생년월일, 국가, 성별, 키/체중)
- Step 2: 생활습관 (흡연, 음주, 운동, 수면, 식습관)
- Step 3: 정신건강 + 가족력 (선택)

상단에 프로그레스 바. 각 스텝 간 부드러운 슬라이드 전환. Step 3에는 "건너뛰기" 옵션.

### 6-4. 로딩 애니메이션

- 모래시계가 뒤집히는 애니메이션
- 또는 심장 박동 모니터 (심전도 라인) 애니메이션
- "당신의 시간을 계산하고 있습니다..."
- 3초 의도적 딜레이 (긴장감 조성)

### 6-5. 결과 페이지

- 첫 화면: 검은 배경 + 큰 카운트다운 숫자만 (미니멀)
- 스크롤 다운하면 상세 결과가 서서히 나타남 (fade in)
- 프로그레스 바, 남은 계절, 남은 주말 등이 순차적으로 로드
- "부모님과 남은 시간"은 별도 섹션으로 감성적 연출
- 전체 톤: 무겁지 않게, 담담하면서 여운이 있는 느낌

### 6-6. 색상 & 톤

- 다크 테마 기본 (검정 + 흰색 텍스트)
- 강조색: 시간이 많으면 초록, 적으면 주황 -> 빨강 그라데이션
- 폰트: 카운트다운은 모노스페이스 (디지털 시계 느낌)
- 나머지는 깔끔한 산세리프

"부모님께 보내기" 클릭 시 카카오톡 공유 문구: "엄마, 우리 같이 보낼 수 있는 시간이 얼마 안 남았대. 이번 주말에 만나자"

이 감성 공유가 40-60대 유입을 만듦. 젊은 층이 부모에게 보내면 부모 세대가 유입 → 부모가 자기 친구에게 공유 → 세대 확산.

    lifereceipt.com/ja/lifespan/      -> 일본어

### 8-2. 로컬라이징 포인트

**기대수명 데이터**: 국가별 WHO 데이터 자동 적용.

**남은 명절/계절 항목 현지화**:

- 한국: 설날, 추석, 벚꽃 시즌
- 미국: Thanksgiving, Christmas
- 일본: 오봉, 하나미
- 브라질: Carnaval

**"부모님과 남은 시간" 문화 보정**:

- 한국/일본: 만남 빈도 낮음 (명절 + 가끔) → 적은 숫자 → 더 충격적
- 남미/남유럽: 만남 빈도 높음 → 절대시간은 많지만 "그래도 유한하다"

**단위**: cm/kg vs ft/lb (IP 기반 자동 전환).

**톤**: 한국/일본은 감성적, 영어권은 담담하고 팩트 기반.

### 8-3. 언어 우선순위

인생 영수증과 동일: 한국어+영어 → 일본어 → 스페인어 → 포르투갈어

## 10. 인생 영수증과의 통합 시너지

### 10-1. 데이터 공유

공통 입력 항목과 각 서비스에서의 활용:

- **생년월일**: 수명 계산기에서 현재 나이 산출 / 영수증에서 생존 일수 산출
- **국가**: 수명 계산기에서 기대수명 기준 / 영수증에서 국가별 통계 상수
- **성별**: 수명 계산기에서 성별 기대수명 / 영수증에서 통계 보정
- **수면 시간**: 수명 계산기에서 수명 보정 / 영수증에서 수면 총량 계산
- **커피**: 수명 계산기에서 미사용 / 영수증에서 커피 소비량
- **스마트폰**: 수명 계산기에서 좌식 시간 보정 / 영수증에서 총사용시간

### 10-2. 교차 유도 흐름

수명 계산기 완료 후: "당신의 인생을 영수증으로도 확인해보세요" (추가 입력 불필요) → 인생 영수증 결과 즉시 표시.

인생 영수증 완료 후: "남은 시간이 궁금하지 않으세요?" (건강 습관 몇 개만 추가 입력) → 수명 계산기 결과 표시.

### 10-3. 통합 공유 카드

## 11. 윤리적 고려사항 & 면책

### 11-1. 필수 면책 문구

결과 페이지 상단과 하단에 표시:

> "이 결과는 통계 데이터 기반의 추정치이며, 의학적 진단이 아닙니다. 실제 수명은 무수한 요인에 의해 결정됩니다. 건강에 대한 우려가 있다면 의료 전문가와 상담하세요."

### 11-2. 하한선 규칙 (재강조)

- **절대 규칙: 예상 수명 = MAX(현재 나이 + 5, 계산된 수명)**
- 어떤 입력 조합이든 "곧 죽는다"는 결과가 나오면 안 됨
- 결과가 국가 평균보다 크게 낮을 경우 톤 조심:
  - (X) "당신의 수명은 짧습니다"
  - (O) "생활습관을 개선하면 수명을 크게 늘릴 수 있어요"

### 11-3. 정신건강 안전장치

- 한국: 자살예방상담전화 1393
- 미국: 988 Suicide & Crisis Lifeline
- 국가별 위기 상담 번호 DB 내장

### 11-4. 연령 제한

- 만 13세 미만은 서비스 이용 불가 (생년월일 입력 시 필터링)
- 이유: 어린이에게 "남은 수명" 보여주는 것은 부적절

## 12. MVP 우선순위

### Week 1: MVP (인생 영수증과 동시 또는 직후)

## 13. 핵심 성공 지표 (KPI)

- **총 계산 횟수**: 목표 200,000+ (1개월). 수명 계산기가 영수증보다 클릭률 높을 것.
- **공유율**: 목표 35%+. 감성 항목이 공유 동기를 강화.
- **인생 영수증 교차 전환**: 목표 40%+. 수명 계산기에서 영수증으로 이동하는 비율.
- **"부모님께 보내기" 클릭률**: 목표 15%+. 감성 바이럴의 핵심 지표.
- **평균 체류 시간**: 목표 120초+. 시뮬레이션 섹션이 체류 늘림.

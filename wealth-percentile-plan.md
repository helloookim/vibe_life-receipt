# Wealth Percentile (글로벌 부 순위) — 서비스 기획서

---

## 1. 컨셉 한 줄 요약

**"전 세계 80억 명 중 당신은 몇 등일까?"**

연소득 또는 자산을 입력하면 전 세계 / 국가별 / 연령별 부의 순위를 퍼센타일로 보여주는 바이럴 웹 서비스.

---

## 2. 핵심 심리 메커니즘

부 순위가 바이럴 되는 이유는 **자기 위치 확인 욕구 + 비교 본능** 때문.

- "나 세계 상위 3%래" → 자부심 → 자랑 공유
- "나 한국에선 하위 40%래" → 자조 → "ㅋㅋ 현실" 공유
- 글로벌 vs 국내 순위 격차가 대화 소재를 만듦: "한국에선 평범한데 세계적으론 상위권이네?"

추가 바이럴 포인트:

- **비교 가능성**: "너는 몇 %야?" → 자연스러운 링크 전달
- **반복 가능성**: 연봉 인상, 이직, 투자 수익 등 변화 시 재방문
- **범용성**: 학생부터 은퇴자까지 누구나 "내 위치"가 궁금함

---

## 3. 입력 항목 설계

**필수 2개 + 선택 4개**, 총 10초 내 완료 목표. 세 서비스 중 가장 빠른 입력.

### 필수 입력

- **연소득 (세전)**: 숫자 입력. 통화는 국가 선택 시 자동 설정. 슬라이더 + 직접 입력 병행.
- **국가**: 드롭다운 (IP 자동 선택). 국가별 소득 분포 데이터 적용.

### 선택 입력 ("더 정확한 순위 보기" 아코디언)

- **나이**: 숫자 입력. 연령대별 소득 비교 활성화.
- **가구 인원수**: 슬라이더 (1-8명). 1인당 소득으로 보정 (OECD 균등화 소득 방식).
- **순자산**: 숫자 입력. 소득이 아닌 자산 기준 순위도 제공.
- **직업군**: 5버튼 (사무직 / 전문직 / 자영업 / 프리랜서 / 학생-기타). 같은 직업군 내 비교.

### 입력 UX 핵심

- **통화 자동 전환**: 한국 선택 시 원화(만원 단위), 미국 선택 시 달러, 일본 선택 시 엔
- **프리셋 버튼**: "최저임금", "중위소득", "상위 10%" 등 빠른 선택 버튼 제공
- **익명성 강조**: 입력 필드 옆에 "데이터는 저장되지 않습니다" 문구 상시 노출
- **소득 입력 심리 허들 낮추기**: 정확한 금액 대신 범위 선택도 가능하게 (3000-4000만원 등)

## 4. 순위 계산 로직

### 4-1. 데이터 소스

**글로벌 소득 분포**

- World Bank PovCalNet / Poverty and Inequality Platform (PIP)
- 전 세계 소득 분포 데이터 (PPP 기준)
- 약 200개국 커버

**국가별 소득 분포**

- OECD Income Distribution Database (선진국 34개국 상세)
- 한국: 통계청 가계금융복지조사
- 미국: US Census Bureau Current Population Survey
- 일본: 총무성 가계조사
- 기타: World Bank, ILO 데이터

**자산 분포**

- Credit Suisse Global Wealth Databook (현 UBS)
- 국가별 자산 분위 데이터

### 4-2. 계산 방식

    글로벌 퍼센타일 = 입력 소득(PPP 환산) 의 전 세계 소득 분포 내 위치

    국가 내 퍼센타일 = 입력 소득의 해당 국가 소득 분포 내 위치

    연령별 퍼센타일 = 입력 소득의 해당 국가 + 연령대 소득 분포 내 위치

### 4-3. PPP (구매력평가) 환산

- 단순 환율 변환은 의미 없음 (한국 5000만원 vs 인도 5000만원의 실질 구매력 차이)
- World Bank PPP conversion factor 적용
- 결과에 "PPP 기준" vs "명목 기준" 토글 제공

### 4-4. 소득 분포 모델링

- 대부분의 국가 소득 분포는 로그정규분포(log-normal)에 근사
- 각 국가별 중위소득 + 지니계수로 분포 추정
- 상위 1% 이상은 파레토 분포로 보정 (극부유층 반영)

### 4-5. 데이터 업데이트

- 연 1회 업데이트 (World Bank, OECD 데이터 갱신 주기에 맞춤)
- JSON 파일로 내장 (API 의존 없이 클라이언트 계산)
- 데이터 기준연도를 결과 페이지에 명시

## 5. 결과 페이지 설계

### 5-1. 메인 결과 (첫 화면)

    +-------------------------------------------+
    |                                           |
    |        YOU ARE RICHER THAN                 |
    |        당신은 전 세계                       |
    |                                           |
    |           96.8%                            |
    |         (큰 숫자, 애니메이션 카운트업)       |
    |                                           |
    |       보다 부유합니다                       |
    |                                           |
    |  80억 명 중 약 77억 4천만 명보다 많이 법니다 |
    |  당신보다 부유한 사람: 약 2억 6천만 명       |
    |                                           |
    +-------------------------------------------+

### 5-2. 상세 비교 항목

**글로벌 순위**

- 전 세계 퍼센타일: 상위 3.2% (96.8th percentile)
- 당신보다 가난한 사람: 약 77.4억 명
- 당신보다 부유한 사람: 약 2.6억 명
- 등수 환산: 전 세계 약 2억 5,600만 등

**국가 내 순위**

- 한국 내 퍼센타일: 상위 42% (58th percentile)
- 한국 중위소득 대비: 112%
- 같은 연령대 내: 상위 35%

**구매력 비교 (PPP)**

- 당신의 소득으로 빅맥을 연간 X개 살 수 있습니다
- 미국 기준 환산 소득: $XX,XXX
- 스위스 기준 환산 소득: CHF XX,XXX

### 5-3. 시각화 요소

**인구 도트맵**

100명의 사람 아이콘 중 당신의 위치를 하이라이트:

    [사람][사람][사람]...[★나][사람][사람]
    (100명 중 97번째 = 상위 3%)

**국가 비교 바**

    글로벌   [================|==] 상위 3.2%
    한국     [=========|=========] 상위 42%
    미국     [===========|=======] 상위 38%

**시간 비교 (흥미 유발)**

- "당신의 연소득을 벌려면": 미국 최저임금으로 X개월 / 인도 평균임금으로 X년
- "당신의 1시간 임금": 세계 중위 시급의 X배
- "당신의 일소득": 전 세계 하위 10%의 월소득과 비슷

### 5-4. "만약에" 시뮬레이션

슬라이더로 소득을 조정하면 실시간으로 순위가 변하는 인터랙티브 기능:

    [내 순위 바꿔보기]

    연봉 5000만원 -> 상위 3.2% (글로벌) / 상위 42% (한국)
    연봉 1억원   -> 상위 1.1% (글로벌) / 상위 12% (한국)
    연봉 3억원   -> 상위 0.3% (글로벌) / 상위 2% (한국)

    "상위 1%가 되려면?": 한국 기준 연 X만원 / 글로벌 기준 연 $X

### 5-5. 감성/인사이트 섹션

결과 하단에 짧은 인사이트:

- "깨끗한 물을 마실 수 있다면 당신은 이미 상위 25%입니다"
- "스마트폰을 가지고 있다면 상위 50%입니다"
- "은행 계좌가 있다면 상위 40%입니다"

이런 기준점들이 시야를 넓혀주면서도 공유 포인트가 됨.

## 6. UI/UX 설계

### 6-1. 전체 흐름

    [랜딩] -> [입력] -> [로딩] -> [결과: 퍼센타일] -> [비교] -> [시뮬레이션] -> [공유]
      1초     5~10초    2초       체류(핵심)          스크롤    인터랙션        바이럴

세 서비스 중 입력이 가장 짧아서 전환율이 높을 것.

### 6-2. 랜딩 페이지

- 배경: 지구 이미지 또는 세계지도 실루엣 (다크 톤)
- 카피: "Where do you stand?" / "80억 명 중 당신은 몇 등?"
- 중앙에 소득 입력 필드 하나만 크게 (랜딩 = 입력 폼 통합)
- 하단: "지금까지 OOO명이 확인했습니다"
- 프리셋 버튼: "최저임금으로 해보기", "평균 소득으로 해보기"

### 6-3. 입력 폼

- **랜딩 페이지에 통합** (별도 페이지 X). 입력이 2개뿐이므로 가능.
- 국가 선택 → 통화 자동 변경 → 소득 입력 → [결과 보기] 버튼
- "더 정확한 순위" 클릭 시 나이/가구원수/자산/직업 아코디언 펼침
- 소득 범위 프리셋 버튼으로 빠른 입력 지원

### 6-4. 로딩 애니메이션

- 지구본이 회전하면서 인구 카운터가 올라가는 애니메이션
- "80억 명 중 당신의 위치를 찾고 있습니다..."
- 2초 딜레이 (짧게 — 입력이 간단한 만큼 로딩도 빠르게)

### 6-5. 결과 페이지

- 첫 화면: 큰 퍼센타일 숫자가 0%에서 카운트업 애니메이션
- "96.8%" 같은 숫자가 올라가면서 멈추는 연출
- 바로 아래: "80억 명 중 77.4억 명보다 부유합니다" 텍스트
- 스크롤 다운: 국가 내 순위, 연령별 순위, 시각화
- 추가 스크롤: 시뮬레이션, 인사이트, 공유

### 6-6. 색상 & 톤

- 다크 블루/네이비 기본 (금융 느낌이면서 세련됨)
- 강조색: 골드 (상위권일 때) / 실버 / 브론즈 (게이미피케이션)
- 퍼센타일 숫자: 큰 산세리프, 볼드
- 전체 톤: 팩트 기반, 중립적, 판단하지 않는 느낌

## 8. 다국어 / 글로벌 전략

### 8-1. URL 구조 (통합 사이트)

    lifereceipt.com/wealth/         -> 영어
    lifereceipt.com/ko/wealth/      -> 한국어
    lifereceipt.com/ja/wealth/      -> 일본어
    lifereceipt.com/es/wealth/      -> 스페인어
    lifereceipt.com/pt/wealth/      -> 포르투갈어

### 8-2. 로컬라이징 포인트

**통화 & 단위**:

- 한국: 만원 단위 (연 5,000만원)
- 미국: 달러 (연 $50,000)
- 일본: 만엔 단위 (연 500万円)
- 유럽: 유로 (연 40,000 EUR)
- 입력 시 천 단위 구분자도 로케일에 맞춤 (1,000 vs 1.000)

**비교 기준 현지화**:

- 한국: "서울 아파트 평균가 대비 연소득 비율", "치킨 몇 마리"
- 미국: "Big Mac Index", "average rent in NYC"
- 일본: "도쿄 맨션 대비", "라멘 몇 그릇"
- 브라질: "salario minimo 몇 배"

**문화적 톤 차이**:

- 한국/일본: 자조적 유머 허용 ("부자라더니 왜 이러고 사나")
- 영어권: 중립적 팩트 기반 ("Here's where you stand globally")
- 남미: 밝고 에너지 있는 톤

**민감도 주의**:

- 저소득 국가 사용자에게 "당신은 하위 X%입니다"는 불쾌할 수 있음
- 항상 긍정적 프레이밍: "당신은 X명보다 부유합니다" (하위가 아닌 상위 기준)
- 글로벌 인사이트 섹션으로 상대적 관점 제공

### 8-3. 언어 우선순위

한국어+영어 → 일본어 → 스페인어 → 포르투갈어 (동일 패턴)

## 9. 수익화

### 9-1. 광고

- 결과 페이지 하단 배너
- 금융 관련 키워드라서 **광고 단가(CPC/CPM)가 세 서비스 중 가장 높음**
- 타겟 광고 카테고리: 투자, 보험, 부동산, 재테크 교육, 은행 상품

### 9-2. 제휴

- 재테크/투자 교육 플랫폼 CPA 광고
- 해외 송금 서비스 (Wise, Remitly 등) — 글로벌 사용자 기반
- 보험 비교 사이트
- 자산관리 앱

### 9-3. 프리미엄 기능

- **상세 리포트**: 국가별/연령별/직업별 심층 비교 ($0.99)
- **연간 트래킹**: 매년 소득 변화에 따른 순위 변동 추적
- **도시별 비교**: 서울 vs 뉴욕 vs 도쿄 등 도시 단위 비교 (데이터 확보 시)

## 10. 인생 영수증 / 수명 계산기와의 통합 시너지

### 10-1. 데이터 공유

- **국가**: 세 서비스 모두 공통
- **나이**: 수명 계산기와 공통
- **소득**: 인생 영수증의 "돈 카테고리"에 활용 가능 (커피 지출 비율, 식비 비율 등)

### 10-2. 교차 유도 흐름

글로벌 부 순위 완료 후: "당신의 시간도 계산해볼까요?" → 수명 계산기로 유도 (소득은 건강과 상관관계가 있으므로 자연스러운 연결).

인생 영수증 완료 후: "지금까지 번 돈은 세계 몇 등일까?" → 부 순위로 유도.

## 11. 윤리적 고려사항 & 면책

### 11-1. 프라이버시

- **소득 데이터 서버 미저장**: 모든 계산은 클라이언트에서 수행. 명시적 안내.
- **공유 시 소득 미노출**: OG 이미지와 공유 링크에는 퍼센타일만 표시, 실제 입력 금액은 절대 포함하지 않음.
- **결과 URL에 소득 미포함**: URL 파라미터에 퍼센타일만 인코딩, 역추적 불가.

### 11-2. 면책 문구

결과 페이지에 표시:

> "이 결과는 공개된 통계 데이터 기반의 추정치입니다. World Bank, OECD 등의 데이터를 사용하며, 실제 소득 분포와 차이가 있을 수 있습니다. 재무 상담이 필요하시면 전문가와 상의하세요."

### 11-3. 민감도 관리

- 하위 퍼센타일 결과도 절대 비하하는 톤을 쓰지 않음
- 항상 "상위 X%" 또는 "X명보다 부유합니다" 형태로 표시
- "소득이 적다"가 아니라 "다른 관점에서 보면" 인사이트 제공
- 극단적 저소득 입력 시에도 긍정적 메시지: "깨끗한 물, 인터넷 접속이 가능하다면 이미 많은 것을 가지고 있습니다"

### 11-4. 데이터 정확성

- 데이터 기준연도 항상 명시 ("Based on World Bank 2023 data")
- "추정치"임을 명확히
- 비공식 경제(informal economy)가 큰 국가의 경우 주의 문구 추가

## 12. 경쟁 분석

### 12-1. 기존 서비스

- **GlobalRichList.com**: 2000년대 만들어진 원조. UX 구식, 모바일 미대응, 업데이트 불명확, 국가별 비교 없음.
- **HowRichAmI (GiveWell)**: 기부 유도 목적. 깔끔하지만 공유 기능 없음, 바이럴 설계 아님.
- **Wealthometer**: 영국 특화. 글로벌 비교 약함.

### 12-2. 차별화 포인트

- **글로벌 + 국가 내 + 연령별 3중 비교**: 기존 서비스는 글로벌만 제공
- **바이럴 최적화**: OG 이미지, 공유 문구, 비교 기능 — 기존 서비스에 전무
- **현대적 UX**: 모바일 퍼스트, 애니메이션, 인터랙티브 시뮬레이션
- **다국어 + 로컬 비교**: 각 국가 문화에 맞는 비유와 비교 기준
- **세 서비스 통합**: 수명 계산기, 인생 영수증과 묶여서 교차 트래픽

## 13. MVP 우선순위

### Week 1: MVP

- 연소득 + 국가 입력 (필수 2개)
- 글로벌 퍼센타일 계산 (World Bank PPP 데이터)
- 결과 페이지: 퍼센타일 + "X명보다 부유합니다"
- 인구 도트맵 시각화
- 한국어 + 영어
- 이미지 저장 + 카카오톡/트위터 공유
- 프라이버시 안내 + 면책 문구

### Week 2: 비교 강화

- 국가 내 퍼센타일 추가
- 연령별 비교 추가 (나이 선택 입력)
- "만약에" 시뮬레이션 (소득 슬라이더)
- 동적 OG 이미지
- 수명 계산기 / 인생 영수증 교차 유도

### Week 3: 글로벌 확장

- 일본어 + 스페인어
- 국가별 비유 로컬라이징 (빅맥, 치킨, 라멘 등)
- 순자산 기준 순위 추가
- 직업군별 비교

### Week 4+: 수익화

- AdSense 연동 (금융 카테고리 고단가)
- 재테크/투자 플랫폼 제휴
- 상세 리포트 유료화

## 14. 핵심 성공 지표 (KPI)

- **총 조회 횟수**: 목표 150,000+ (1개월). 입력이 간단해서 전환율 높을 것.
- **공유율**: 목표 40%+. "너 몇 %야?" 비교 동기가 강력.
- **교차 서비스 전환**: 목표 30%+. 부 순위에서 수명 계산기 또는 영수증으로 이동.
- **평균 체류 시간**: 목표 90초+. 시뮬레이션과 비교 섹션이 체류 유도.
- **광고 수익 단가**: 세 서비스 중 최고 CPM 예상 (금융 키워드).

## 15. 기술 참고사항

### 15-1. 데이터 크기

- 200개국 소득 분포 데이터: 약 50-100KB (JSON 압축 시)
- PPP 환산 계수: 약 10KB
- 전체 데이터를 클라이언트에 내장 가능 (API 불필요)

### 15-2. 계산 성능

- 로그정규분포 CDF 계산은 JavaScript로 충분 (수학 라이브러리 불필요)
- 클라이언트에서 즉시 계산, 서버 부하 0

### 15-3. 데이터 수집 자동화

- World Bank API에서 연 1회 데이터 갱신 스크립트
- OECD.Stat에서 국가별 상세 데이터 수집
- 갱신 시 JSON 파일만 교체하면 됨
